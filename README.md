# PageModule
Unity UGUI에서 사용 가능한 Page System

## 핵심 기능
사용자가 방문한 Page를 다시 돌아올 수 있도록 구현

## 역할 정리
1. BasePage
    - PageUI의 컴포넌트
    - 페이지 이동 및 돌아오기 이벤트 컨트롤
    - 페이지 껐다 키기
2. PageStack
    - 사용자가 방문한 페이지 저장 및 불러오기
